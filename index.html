Privacy Policy - Hakka Calculator

Privacy Policy for Hakka Calculator (com.hakka.cal)

Effective Date: 17/09/2024

Welcome to Hakka Calculator! Your privacy is important to us, and we are committed to protecting your data while using our application. This Privacy Policy explains how we handle your data, what technologies we use, and why we use them. By using the Hakka Calculator, you agree to the terms outlined in this policy.

1. Introduction

Hakka Calculator is a functional and easy-to-use calculator app designed to perform various mathematical calculations, including basic arithmetic, trigonometric functions, logarithmic calculations, and more. This policy details how the app operates, what data it collects (if any), and how we ensure your privacy is safeguarded.

2. Data Collection and Use

Hakka Calculator does not collect, store, or transmit any personal data from its users. All calculations are performed locally on your device without any external data transmission. We do not collect or share your input or output data.

3. Key Features and Functions

  • Basic Arithmetic Operations: Addition, subtraction, multiplication, and division.
  • Advanced Mathematical Functions: Includes trigonometric functions (sin, cos, tan), logarithmic calculations (log, ln), square roots, powers, and factorials.
  • Dynamic UI Adjustments: The app adjusts the font size dynamically based on the input length to ensure readability.
  • Error Handling and Notifications: The app provides user-friendly notifications for invalid inputs or unsupported operations, such as large factorial calculations.
  • No External Permissions Required: The app does not request any permissions, ensuring no access to personal data, location, contacts, or any other sensitive information.

4. Technology Stack

Hakka Calculator is built using the following technologies:

  • Programming Language: Kotlin
  • Development Environment: Android Studio
  • Libraries and Dependencies:
    • AndroidX Libraries: Used for modern UI components and enhanced app compatibility.
    • Material Design Components: Used for UI consistency and aesthetic appeal.
    • BigInteger (Java Math Library): Used for handling large numbers, particularly in calculations involving large factorials.

5. SDK Usage Information

Hakka Calculator uses the following SDKs to provide functionality, improve user experience, and ensure optimal performance:

1. AndroidX SDK

Purpose: AndroidX is a set of libraries that are used to provide backward-compatible versions of Android framework components.

Why We Use It:

  • To ensure a consistent and modern user experience by providing updated UI components like ConstraintLayout and CardView.
  • To enhance compatibility across various devices by using standardized components that are actively maintained and updated.
  • It provides essential libraries like AppCompatActivity to maintain the compatibility of modern UI elements across different Android devices.

2. Material Components SDK

Purpose: Material Components for Android provide the foundation for implementing Material Design, Google's design system that supports modern and beautiful UI.

Why We Use It:

  • To create a visually appealing interface that is user-friendly and consistent with contemporary app design standards.
  • Material components like Button, CardView, and TextView enhance the app’s look and feel, making it more interactive and engaging for the user.
  • These components help in maintaining a consistent design language throughout the app, providing a cohesive user experience.

3. Kotlin SDK

Purpose: Kotlin is a modern programming language that simplifies Android app development with enhanced syntax, safety features, and interoperability with Java.

Why We Use It:

  • Kotlin allows us to write safer, more concise code with fewer chances of errors, enhancing the overall performance of the app.
  • The language’s null safety features significantly reduce common programming errors, making the app more stable and less prone to crashes.
  • It integrates seamlessly with existing Java libraries, enabling the use of robust mathematical libraries for calculations.

4. Java Math Library (BigInteger)

Purpose: The Java Math library provides BigInteger, a class designed to handle calculations involving large numbers that exceed the capacity of primitive data types.

Why We Use It:

  • Used specifically for performing operations like factorial calculations of large numbers to avoid overflow issues that can occur with standard data types like Int or Long.
  • Ensures accurate results for advanced mathematical computations, making the app capable of handling complex calculations without performance degradation.

6. Permissions

Hakka Calculator does not request or use any permissions on your device. This includes:

  • No Access to Personal Data: We do not access or store your personal information.
  • No Location Tracking: The app does not track or use your location.
  • No Internet Access: Hakka Calculator does not require or use an internet connection, ensuring all operations remain local to your device.
  • No Access to Contacts, Camera, or Files: The app does not access any of your contacts, camera, photos, files, or any other personal data.

7. Error Handling and Notifications

Hakka Calculator includes robust error handling mechanisms to guide users during calculations:

  • Division by Zero: Notifies users that division by zero is not allowed.
  • Invalid Inputs: Displays error messages for invalid formats, such as incorrect decimal usage or unsupported characters.
  • Large Factorial Calculations: Notifies users when factorial calculations exceed practical limits (e.g., factorials of numbers greater than 20).
